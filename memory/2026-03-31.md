### 🤖 Rust Backend (tutorial-generator/)
- **Complete CLI application** with argument parsing via `clap`
- **Git analysis** using libgit2 (`git2` crate) with vendored OpenSSL
- **Commit history extraction** with rich metadata (author, date, message)
- **File change detection** - detects added/modified/deleted/renamed files
- **Language detection** - identifies 30+ programming languages from file extensions
- **Intelligent sectioning** - groups commits by logical boundaries (new features, refactors)
- **Markdown generator** - produces beautiful, structured tutorials
- **JSON output support** - for programmatic processing
- **Cloning support** - can analyze both local paths and remote URLs (auto-clone)
- **Configurable** - title, description, max commits, output format all tunable
